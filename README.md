# Python_Basic_Course
This repository consists of lectures and assignments for Basics of Python course

### Course Outline

| Module | Description |
|---|----|
| Module 00  | Installing Python – Jupyter Notebook Comments, Print |
| Module 01  | Variables – naming, assignment, scope, input |
| Module 02  | Data Types – string, numeric, sequence, mapping, set, boolean, binary, none, casting |
| Module 03  | Condition Statement – IF Else, Switch (Match) |
| Module 04  | Loops – While and For loops |
| Module 05  | Functions – arguments, scope, lambda, recursion |
| Module 06  | Built-In functions – range, arrays, iterators, modules, date, math, pip |
| Module 07  | Exceptions (Try…..Expect) and string formatting |
| Module 08  | File Handling |
| Module 09  | Oops – classes and objects, init method |
| Module 10  | Oops – property, methods |
| Module 11  | Oops – polymorphism, Encapsulation, Inner Classes |
| Module 12  | Python modules – Numpy, Pandas, SciPy |
| Module 13  | Matplotlib, Image Modules |
